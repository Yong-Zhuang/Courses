HTML5

<section/>
This element defines a section in HTML document, so I can put defferent elements into it. e.g. I my logo, my motto, my photo and my profile in a section "personal-intro".
<header/>
This element defines a header for the document or a section. In my home page, I put my logo and my motto in the header of section "personal-intro".
<footer/>
This element defines a footer for the document or a section. In my home page, I put "Valid XHTML 1.0 Strict" icon, "Valid CSS" icon and copyright information in it.
<nav/>
This element defines navigation links in the document. In my home page, I put all quick links in this element.
<aside/>
This element defines content aside from the page content. In my home page, I want the navigation lie on the right side, so I put the element nav in it.

CSS

min-width and min height
Using min-width and min height to ensure the element has a minimal size. In my website, some page has few context, so these css can make this page in a resonable way for browsering.
a:link，a:hover, a:focus, a:active and a:visited
Using these css to make links with different states in disprate way. In my website, I design different font color for a:link, a:hover, a:focus, a:active and a:visited, so user can recognize which page has been browsered easily.
.sidebar li
Using this css to only change the style of element <li> in the <div> "sidebar". So the elements <li> in other elements still have their own styles.
top, left and position
Using these css to put a element to the place you want it be. In my website, I use these css to put the "red torii" at the top right part.
@media
@media rule allows different style rules for different media in the same style sheet. In my website, I use this css to make sure the height and width of main context fit for handheld machines.
box-shadow
Using this css to attache drop-shadows to a element. In my website, I use this css to create a shadow to my profile box.
:before
Using this css to insert content before a element. In my website, I use this css to create the background word "Yong" before main context.
z-index: -1
Using z-index property specifies the stack order of an element. In my website, I use "z-index: -1" to make sure the background word "Yong" below the layer of main context.
@-webkit-keyframes
Using this css we can create animations which can replace Flash animations, animated images, and JavaScripts in existing web pages. In my website, I use this css to rotate and fade the background word "Yong".
